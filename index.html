заголовок 2 уровня

HTML - структура, основа каждой страницы и веб-сайта. Язык гипертекстовой разметки
CSS - стили и офомление (цвет, размер, расстояние)
JS - интерактивность, набор функций, все действия пользователя на сайте
Теги (красные) - это тот синтаксис, с помощью которого записываем html элементы.
Строчные - жирный, зачёркнутый... - выделяют часть текста, принося ему определённое (н-р, визуальное) свойство. Занимают размер только своего контента и устраиваются друг за другом. Можем размещать внутри других.
Блочные - параграфы, заголовки, дивы, списки - занимают всю ширину видимой части браузера. Все элементы после них переносятся на следующую строку.
Атрибуты (салатовые) - есть индивидуальные для одного (href) или общие для всех тегов - классы (стилизирование элементов), индификаторы (выделение конкретных элементов среди других и навигции по документу).
